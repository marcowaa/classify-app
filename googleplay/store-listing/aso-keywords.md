# ASO Keywords Strategy — Classify

## الاستراتيجية العامة

Google Play يفهرس: الاسم (أعلى وزن) + الوصف القصير + الوصف الطويل (كامل)
الكثافة المثلى: 2-3% للكلمات الرئيسية، 1-2% للثانوية
أهم المواضع: أول 180 حرف + آخر 180 حرف من الوصف الطويل

---

## الكلمات المفتاحية العربية

### الفئة الأولى — أولوية عالية (مُدرجة في الاسم والوصف القصير)
| الكلمة | التكرار في الوصف الطويل | الهدف |
|---|---|---|
| تعليم / تعليمي / تعليمية | 8-10 مرات | #1 keyword |
| أطفال / طفلك / الأطفال | 8-10 مرات | #2 keyword |
| رقابة أبوية | 3-4 مرات | تمييز تنافسي |
| ألعاب تعليمية | 4-5 مرات | الألعاب |
| مهام تعليمية | 4-5 مرات | فريد لـ Classify |

### الفئة الثانية — أولوية متوسطة (في الوصف الطويل)
| الكلمة | التكرار |
|---|---|
| شجرة النمو / نمو | 5-6 مرات |
| نقاط ومكافآت | 3-4 مرات |
| متابعة الأطفال | 2-3 مرات |
| تحفيز الطفل | 2-3 مرات |
| سوق معلمين | 2-3 مرات |
| منصة تعليمية | 2-3 مرات |
| تحكم أبوي | 2-3 مرات |

### ذيل طويل (Long-tail) — نية شراء أعلى
```
تطبيق مهام تعليمية للأطفال
برنامج رقابة أبوية عربي
ألعاب تعليمية رياضيات عربية

منصة تعليمية للأسرة العربية
تطبيق تعليمي إسلامي للأطفال

```

## خطة التوزيع في النصوص

### الاسم (30 حرف) — يحتوي:
- "تعليم" ✅
- "تحفيز" ✅  
- "الأطفال" ✅

### الوصف القصير (77/80 حرف) — يحتوي:
- "مهام وألعاب" ✅
- "رقابة أبوية" ✅
- "شجرة نمو" ✅
- "مكافآت" ✅
- الفعل "علّم" (action-oriented) ✅

### الوصف الطويل — أول 180 حرف يحتوي:
- "تعليم"، "أطفال"، "رقابة أبوية"، "ألعاب تعليمية"، "شجرة نمو" ✅

### الوصف الطويل — آخر 180 حرف يحتوي:
- "Classify"، "التعلم"، "التربية"، CTA قوي ✅

## تحديثات دورية

- كل 4-6 أسابيع: راجع Acquisition Reports في Play Console
- ابحث عن كلمات بحث المستخدمين الفعلية وأضف الجديد منها
- في المواسم الدراسية (سبتمبر، يناير): حدّث الوصف بكلمات موسمية
- بعد كل ميزة جديدة: أضف كلمتها المفتاحية للوصف
